# PearTech

E-commerce téléphonie & high-tech, réalisé en autonomie dans le cadre de ma formation DWWM.
Architecture JS orientée composants — code réutilisable et maintenable, sans framework.

## Stack

HTML5 · CSS3 · JavaScript ES6+ · Git

## Fonctionnalités

**Catalogue**
- Filtrage combinable : marque, prix, OS, stockage — mis à jour dynamiquement
- Tri des résultats (prix croissant/décroissant, nouveautés) sans rechargement

**Fiches produits**
- Galerie d'images, caractéristiques techniques, avis utilisateurs
- Indicateurs de confiance : disponibilité stock et délai de livraison

**Panier & commande**
- Tunnel de commande simplifié : panier → paiement → confirmation
- Gestion des favoris

**Compte utilisateur**
- Profil utilisateur

**Légal & support**
- CGU, CGV, mentions légales, FAQ, contact
